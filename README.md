# InteractiveMS Flutter TV App

Simple Flutter Android TV app for Mi Box.

## Build command for Mi Box

```bash
flutter build apk --release --target-platform android-arm64
```

The APK will be created at:

```text
build/app/outputs/flutter-apk/app-release.apk
```

## Codemagic

Upload this whole project to GitHub, connect it to Codemagic, then build an Android APK.
